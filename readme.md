# SEDics2 #

SEDics2是一款通过社工信息的元素来生成爆破字典的工具。前身为SEDcis，添加了图形界面更方便操作，优化了字典生成的算法，在生成大型字典的时候速度更快。

特点

1.生成字典的规则文件可通过自身的经验和理解不断补充，使其更适合自身常见的场景。

2.可对字典的规则，组成元素设置大小梯度，准确控制字典的大小以匹配实战环境。

3.可自由添加不同的规则生成不同类型的字典，如 密码，用户名，后台路径等。
